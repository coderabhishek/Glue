The list of Dependencies in order to run this "Python to C++" translator is :

1.The translator requires a version of ubuntu14.04 or higher .
2.The translator functions on g++4.8.4 version or higher.



To build/modify the translator:

1.In order to modify the code and then test it, the device should have googletest-1.7.0 version installed in it.
2.Developers are expected to have doxygen running on their system.
